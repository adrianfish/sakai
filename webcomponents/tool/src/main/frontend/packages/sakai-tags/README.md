# @sakai-ui/sakai-tags

The Sakai tags components

## Description

These components provide tags management functionality for Sakai users.

## Installation

```bash
npm install @sakai-ui/sakai-tags
```

## Usage

```javascript
import '@sakai-ui/sakai-tags/sakai-tag-collections.js';

// In your HTML
<sakai-tag-collections></sakai-tag-collections>
```

## Linting and formatting

To scan the project for linting and formatting errors, run

```bash
npm run lint:fix
```

## Testing with Web Test Runner

To execute a single test run:

```bash
npm run test
```
